# KS-Loading

Enkel laddningsskärm efterliknar den på FBRP.
<br>
Körs på ~ 0.00 ms.
<br>

# Installation

* **VIKTIGT: Ändra inget om du inte vet vad du gör.**

# Funktioner
* Den laddar
* Bytbar logga 
* Anpassningsbar bakgrundsfärg
* Anpassningsbar musik

* CSS Färg Generator: https://cssgradient.io/
